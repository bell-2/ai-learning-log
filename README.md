# ai-learning-log

AI 활용 / Claude Code 학습 로그.

AI와 대화하며 배운 것을 **그날그날 실시간으로 기록**하는 살아있는 노트입니다.
[mykim-aus/ml-dl-learning-log](https://github.com/mykim-aus/ml-dl-learning-log)의 방식(개념)을 차용했습니다.

## 원칙 (Andrej Karpathy, "전문가가 되는 법")

1. 프로젝트를 깊이 있게 **끝까지** 한다
2. 배운 것을 **자기 언어로 정리**한다
3. 남이 아니라 **과거의 나**와만 비교한다

## 이 노트의 규칙

- 이해 못 한 채로 넘어가지 않는다 — 질문이 꼬리를 물면 그대로 기록한다
- 추상 설명보다 **실제로 해보고 확인한 것**을 적는다
- 실패·삽질도 기록한다 (원인을 파고든 과정이 제일 남는다)
- 자주 헷갈리는 지점은 ⚠️ 로 표시한다
- 새 내용은 항상 파일 끝에 **추가(append)** 한다 — 과거 기록은 고치지 않는다
